# j-

Information about Jamie Hoeks

## Usage

### Local

Install with npm...

```sh
$ npm install --save j-
```

..then include into your project.

```js
const info = require('j-');
console.log(info.name); // => "Jamie Hoeks"
```

### Global

Install with npm...

```sh
$ npm install --global j-
```

...then run `j-` in your terminal (with optional argument).

```sh
$ j-
{ name: 'Jamie Hoeks',
...
$ j- name
Jamie Hoeks
```
